## 프로젝트 소개

총 10가지의 stage가 제공되며 각 stage마다 선택지가 존재합니다.
사용자가 선택한 선택지를 기준으로 ChatGPT가 사용자에게 어울리는 해리포터 캐릭터를 추천해줍니다.
사용자의 해리포터 캐릭터와 대비하여 어울리는 캐릭터, 어울리지 않는 캐릭터 또한 추천해줍니다.

## 사용기술

- Next.js (ver.13)
- styled-component
- TypeScript
- React.js
  - Recoil
- htmlToImage
- openai
